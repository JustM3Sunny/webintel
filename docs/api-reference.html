üñ•Ô∏è CLI Commands

webintel search

Perform intelligent web research with AI analysis

webintel search "your query" [options]

Options:

--max-results, -n

Number of sources to analyze (default: 20)

webintel search "AI trends" --max-results 5
--format, -f

Output format: rich, json, markdown, plain (default: rich)

webintel search "news" --format json
--save, -s

Save results to file

webintel search "data" --save
--output-dir, -o

Output directory for saved results

webintel search "research" --output-dir ./results

webintel config

Manage WebIntel configuration

webintel config set-api-key YOUR_API_KEY
webintel config show

webintel --version

Display version information

webintel --version
