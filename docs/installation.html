Installation Guide

Get WebIntel up and running in minutes

Quick Installation

1. Install via pip

pip install webintel

This will install WebIntel and all required dependencies.

2. Set up API Key

export GEMINI_API_KEY="your-gemini-api-key"

Get your free API key from Google AI Studio

Note: The Gemini API is free for personal use with generous rate limits.

3. Test Installation

webintel search "test query"

Run a test search to verify everything is working correctly.

System Requirements

Minimum Requirements

  • Python 3.8 or higher
  • 2GB RAM
  • Internet connection
  • 50MB disk space

Recommended

  • Python 3.10+
  • 4GB+ RAM
  • High-speed internet
  • SSD storage

Platform-Specific Instructions

Windows

Using Command Prompt:

pip install webintel
set GEMINI_API_KEY=your-api-key
webintel search "test"

Using PowerShell:

pip install webintel
$env:GEMINI_API_KEY="your-api-key"
webintel search "test"

macOS

# Install using pip
pip3 install webintel

# Set environment variable
export GEMINI_API_KEY="your-api-key"

# Add to ~/.zshrc or ~/.bash_profile for persistence
echo 'export GEMINI_API_KEY="your-api-key"' >> ~/.zshrc

# Test installation
webintel search "test"

Linux

# Install using pip
pip3 install webintel

# Set environment variable
export GEMINI_API_KEY="your-api-key"

# Add to ~/.bashrc for persistence
echo 'export GEMINI_API_KEY="your-api-key"' >> ~/.bashrc
source ~/.bashrc

# Test installation
webintel search "test"

Using Virtual Environment (Recommended)

Using a virtual environment is recommended to avoid conflicts with other Python packages.

Create Virtual Environment

# Create virtual environment
python -m venv webintel-env

# Activate (Windows)
webintel-env\Scripts\activate

# Activate (macOS/Linux)
source webintel-env/bin/activate

Install WebIntel

pip install webintel

Deactivate When Done

deactivate

Troubleshooting

Common Issues

Permission Denied Error

If you get permission errors during installation:

pip install --user webintel

API Key Not Found

Make sure your API key is properly set:

echo $GEMINI_API_KEY

Command Not Found

If 'webintel' command is not found, try:

python -m webintel.cli search "test"
