üîë API Key Setup

1. Get Your API Key

WebIntel requires a Google Gemini API key for AI-powered analysis. Get your free API key from Google AI Studio.

2. Set Your API Key

Linux/macOS:

export GEMINI_API_KEY="your-api-key-here"

Windows:

set GEMINI_API_KEY=your-api-key-here

Using WebIntel CLI:

webintel config set-api-key YOUR_API_KEY

‚úÖ Verify Installation

# Check version
webintel --version
# Test basic search
webintel search "hello world"

Success!

If you see version information and can run a search, WebIntel is properly installed.

üîß Troubleshooting

Command not found: webintel

Try:

python -m webintel.cli search "test"

Permission denied errors

Use user installation:

pip install --user webintel

API key errors

Make sure your GEMINI_API_KEY environment variable is set correctly and the key is valid.
